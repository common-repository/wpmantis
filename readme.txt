== Description ==
This plugin generates automatically links to your Mantis Bugtracker into your posts.

Related Links:

* <a href="http://twitter.com/MountainGrafix">MountainGrafix @Twitter</a>


== Installation ==

1. Download the Pluginfile
2. Unzip the File on your local PC
3. Copy Folder /wpmantis in your Wordpress Plugin Directory
4. Activate the Plugin through the 'Plugins' menu in WordPress
5. Manage your Settings here: WP Admin -> Settings -> WPMantis
